# LLM Develop Project

LLM 개발을 위한 프로젝트입니다.

## 프로젝트 구조

```
llm_develop/
├── data/               # 데이터 저장 디렉토리
│   ├── raw/           # 원본 데이터
│   └── processed/     # 전처리된 데이터
├── docs/              # 문서
├── notebooks/         # Jupyter 노트북
├── src/               # 소스 코드
└── tests/             # 테스트 코드
```

## 설치 방법

1. 저장소 클론
```bash
git clone [repository-url]
cd llm_develop
```

2. 의존성 설치
```bash
conda env update --file environment.yml
```

## 사용 방법

프로젝트 사용 방법은 추후 업데이트 예정입니다.
